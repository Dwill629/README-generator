
    # Good README Generator

    # Description

    This project is to help with providing a simple ReadMe with information about your application. What the app is for, how to use the app, how to install it, who made contributions and so forth. This is a command-line application that runs with Node.js that dynamically genrates the prompted information into the README file.

    ## Installation
    
    To install the proper dependencies, please use the following command:
    ```
    To generate your own README, first run npm install inquirer...in order to install the npm package.json
    ```
    
    ## Usage
    
    When you use node index.js in your terminal, you will be prompted with questions for your own README.
